# MonkeyBusiness
e-amusement server using FastAPI and TinyDB

for experimental testing and playing


## Instructions:
`pip install -U -r requirements.txt`

`python pyeamu.py`

Edit services url and enable url_slash


## Playable Games:
- `DDR A20 PLUS`
- `DDR A3`
- `IIDX 20 tricoro`
- `IIDX 29 CastHour`
- `SDVX 6 Exceed Gear`
